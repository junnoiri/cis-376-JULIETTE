Advanced Search

I fixed the following points.

  • I intentionally prepared the same text that was at the top of the previous form to test if the previous UI issues were resolved.The previous UI had an issue with form elements overlapping text when clicking on a link that read "Advanced Research". So I adjusted the form to stretch downwards and the problem was solved.

  • By making the design of the course registration form card-shaped, even if the width of the window is changed, the UX will not collapse and the responsive state will be maintained. In addition, I created a modern design based on the UNA base color.

  • In addition, I used Bootstrap's library called Clock Picker for the time clock UI to create a UI that allows users to specify the time more intuitively.

  • In order to create the actual course registration functions (search function, search filter function, etc.), it is necessary to manage user information in a database, and this assignment is more focused on the UI, so I omitted functions to manage information from the search form.

Conclusion

A good UI improves user experience, provides usability and efficiency, is visually appealing, provides error response and feedback appropriately, and increases brand value. I realized once again that the UI is the point of contact with the user and is an important element that should be given proper attention in the development process.
